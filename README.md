# C# and .NET Course Projects

## Overview

This repository contains a collection of **C# and .NET projects** completed as part of the **Advanced Software & Web Developer Diploma** at **Pitman Training**.  
The projects range from beginner console applications to full ASP.NET MVC web applications and demonstrate progressive learning across core programming, object-oriented design, databases, and web development.

Each project was created to reinforce a specific concept and follows structured, incremental learning.

---

## Technologies Used

- C#
- .NET Framework / .NET Core
- ASP.NET MVC
- Entity Framework (Database First & Code First)
- SQL Server (LocalDB)
- Visual Studio
- Git & GitHub
